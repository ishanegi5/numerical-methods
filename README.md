# 📊 Numerical Methods using Python

This repository contains simple and easy-to-understand Python implementations of important **Numerical Methods**.
These programs are written in a beginner-friendly way, making them suitable for **students, practical files, and quick revision**.

---

## 🚀 Methods Implemented

### 🔹 Root Finding Methods

* Bisection Method
* Regula Falsi (False Position Method)
* Newton-Raphson Method

### 🔹 System of Linear Equations

* Gauss Elimination Method (with Partial Pivoting)
* Jacobi Method
* Gauss-Seidel Method

### 🔹 Interpolation Methods

* Newton Forward Interpolation
* Newton Backward Interpolation

## 📌 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/ishanegi5/numerical-methods-python.git
   ```

2. Navigate to the folder:

   ```bash
   cd numerical-methods-python
   ```

3. Run any file:

   ```bash
   python filename.py
   ```

## ⚠️ Notes

* Some methods use fixed iterations for simplicity.
* For better accuracy, tolerance-based stopping can be used.
* Gauss Elimination includes partial pivoting to handle zero pivot cases.
